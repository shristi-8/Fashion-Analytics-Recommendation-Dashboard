

👗 Fashion Analytics & Recommendation Dashboard
📌 Overview

This project is an interactive Power BI dashboard built to analyze customer behavior, fashion trends, purchasing patterns, and recommendation priorities. It helps businesses understand customer preferences, evaluate product performance, and make data-driven decisions through interactive visualizations.
